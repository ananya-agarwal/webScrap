# webScrap

1. https://books.toscrape.com/
2. https://www.amazon.com/s?k=books&i=stripbooks-intl-ship&rh=n%3A283155%2Cp_n_publication_date%3A1250226011&dc&crid=KHQVGNF0L67S&qid=1695451079&rnid=1250225011&sprefix=boo%2Cstripbooks-intl-ship%2C297&ref=sr_pg_1
